# 📚 CloudClass – Smart Classroom & Timetable System

CloudClass is a Django-based web application designed to manage classrooms and timetables efficiently. It provides an admin panel to create, update, and organize classroom schedules in a structured and user-friendly way.

## 🚀 Features

* Classroom Management
* Timetable Scheduling
* Admin Dashboard (Django Admin)
* Structured Database Handling

## 🛠️ Tech Stack

* Django (Python)
* HTML, CSS, Bootstrap
* SQLite
* Git & GitHub
* Render (Deployment)

## ⚙️ Run Locally

```bash
git clone https://github.com/your-username/cloudclass.git
cd cloudclass
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

## 🌐 Live Deployment

Deployed on Render using Gunicorn and Django.
